Math Input Tips

  • Inline math: use a single-dollar wrapper around the expression (example: x^2 + 3x = 0).
  • Display math: use a double-dollar wrapper on its own line (example: a/b = c).
  • Unicode symbols (2, √, π, <=, α, ∫, etc.) are automatically converted to LaTeX
  • Tip: Type equations as plain text in Word/Docs before exporting to PDF.
