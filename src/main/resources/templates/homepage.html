Guidelines
Three Options:
  1. Single PDF: Questions with "Answer: ..." lines included
  2. Two PDFs: Upload questions PDF and a separate answer key PDF
  3. CSV Format: Columns: Difficulty, Question_Text, Choice_1 ... Choice_N, Correct_Answer
Mathematical Equations in PDF:
  • Inline math: wrap in $...$, for example $x^2 + 3x = 0$
  • Display math: wrap in $$...$$, for example $$\frac{a}{b}=c$$
  • Unicode symbols (2, sqrt, pi, <=, alpha, integral, etc.) are automatically converted to LaTeX
  • Tip: Type equations as plain text in your Word/Docs file using $...$ before exporting to PDF
